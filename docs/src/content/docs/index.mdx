import { Card, CardGrid, LinkCard } from '@astrojs/starlight/components';

LMAO is a structured tracing and observability library for TypeScript. Instead of formatting log strings, your code
records **spans** into typed, columnar [Apache Arrow](https://arrow.apache.org/) buffers. Writing a span attribute is a
direct typed-array write on the hot path; string encoding and Arrow conversion happen later, off that path.

<CardGrid stagger>
  <Card title="Typed, structured" icon="setting">
    Define a schema with `defineLogSchema`, then tag spans with typed columns instead of stringly-typed log lines.
  </Card>
  <Card title="Fast hot path" icon="rocket">
    Span writes are direct column writes. Dictionary building and Arrow conversion run on the cold path.
  </Card>
  <Card title="Test by tracing" icon="approve-check">
    Run code that emits spans and assert on the resulting span tree — not just return values.
  </Card>
  <Card title="Runs where you run" icon="laptop">
    Node (`process.hrtime`) and browser/Workers (`performance.now`) entry points, with SQLite and D1 sinks.
  </Card>
</CardGrid>
