`convertSpanTreeToArrowTable` turns a root span buffer (and its children) into a single Apache Arrow table with shared
dictionaries. You get the buffer from a tracer that retains completed traces: `TestTracer` collects them in
`tracer.rootBuffers`, and `ArrayQueueTracer` queues them in `tracer.queue` (or `tracer.drain()`).

:::caution
Get the buffer from the tracer — don't stash `ctx.buffer` in a variable outside the op. An op is a reusable
definition that may run concurrently, so a shared variable races; and once the trace completes the buffer strategy
may recycle the buffer, so reading it later is unsafe.
:::

## Printing and exporting from one trace

You don't have to choose between human-readable output and an Arrow export. `CompositeTracer` fans a single trace run
out to several tracers, so `StdioTracer` can print the span tree while `ArrayQueueTracer` collects the buffers to
convert:

```ts
const tracerOptions = { bufferStrategy: new JsBufferStrategy(), createTraceRoot };

const stdio = new StdioTracer(opContext, tracerOptions);
const queued = new ArrayQueueTracer(opContext, tracerOptions);
const tracer = new CompositeTracer(opContext, { ...tracerOptions, delegates: [stdio, queued] });

await tracer.trace('process-items', processItems, 'user-123');

// drain() hands over the queued root buffers and empties the queue.
const tables = queued.drain().map((rootBuffer) => convertSpanTreeToArrowTable(rootBuffer));
```

See `packages/lmao/examples/arrow-export.ts` for a runnable version.

```ts
import {
  JsBufferStrategy,
  TestTracer,
  convertSpanTreeToArrowTable,
} from '@smoothbricks/lmao';
import { createTraceRoot } from '@smoothbricks/lmao/node';

const tracer = new TestTracer(opContext, {
  bufferStrategy: new JsBufferStrategy(),
  createTraceRoot,
});

await tracer.trace('greet', greet);

const table = convertSpanTreeToArrowTable(tracer.rootBuffers[0]);

console.log(table.numRows); // total rows across all spans in the trace
console.log(table.names); // column names (your schema fields + system columns)
const userIds = table.getChild('userId'); // a single column
```

For a single buffer without its child tree, use `convertToArrowTable(buffer)`.

:::tip
Inside an op you can also reach the current span's root buffer directly through the public `ctx.buffer` handle, then
convert it once the trace completes — handy when you aren't using a retaining tracer. See
[`examples/arrow-export.ts`](https://github.com/smoothbricks/codebase/tree/main/packages/lmao/examples/arrow-export.ts).
:::

:::note
Conversion runs on the cold path — it does not affect the cost of recording spans. For why this split exists, see
[Why columnar tracing](/concepts/columnar-tracing/).
:::
