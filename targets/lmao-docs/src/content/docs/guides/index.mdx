## Runnable examples

The patterns above are demonstrated end-to-end in the package's
[`examples/`](https://github.com/smoothbricks/codebase/tree/main/packages/lmao/examples) directory. Each file is a
self-contained script you can run with Bun:

```sh
bun run examples/basic-usage.ts
```

| Example | Shows |
|---|---|
| `basic-usage.ts` | Schema, feature flags, fluent tag chaining, a child span |
| `fluent-result-api.ts` | `ok`/`err` lifecycle, typed error codes, exceptions |
| `chaining-showcase.ts` | Heavy tag chaining, `with()`, flag-guarded child spans |
| `complete-example.ts` | Scoped attributes, async flags, nested spans, `FlushScheduler` |
| `middleware-pattern.ts` | Request scope set once, inherited by every operation |
| `library-integration.ts` | Composing libraries via `deps` + `prefix` |
| `arrow-export.ts` | Converting a span tree to an Apache Arrow table |
